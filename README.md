# Bootcamp Ver.2

<b>With TypeScript & NestJS & Prisma ORM</b>

## Description

Originally, <b>Bootcamp.ver1</b> was implemented with Javascript & ExpressJS & pure MySQL Queries

New version <b>Bootcamp.ver2</b> will be implemented with Typescript & NestJS & Type ORM

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
